# Lithen Functions - Signals

Signals are simple implementations of a `Observer` / `Publisher and Subscriber` Design Pattern. In which we have, the `DataSignal`, the `DataSignalRecord` and the `Shell Function`. They are used to simplify value updates and reactions to it using the library.

Check more about which of them.

- [DataSignalRecord](./data-signal-record.md)
- [shell](./shell.md)
